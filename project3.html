End-to-End Pizza Management System

I crafted an Entity-Relationship (ER) model using Crow's Foot notation, meticulously designing a comprehensive database tailored to the operational needs of a pizza company. This ER model served as the blueprint for the subsequent development of a MySQL database, where I engineered the SQL code to align perfectly with the ERD. The result was a robust and functional database system that effectively captured and organized the company's data. To further enhance the company's operational efficiency, I developed a Java application that seamlessly interfaced with the MySQL database. This application not only facilitated smooth data transactions but also optimized various operational aspects, streamlining processes and contributing to enhanced efficiency across the board.
